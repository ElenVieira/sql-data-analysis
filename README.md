# SQL Data Analysis Portfolio

This repository contains SQL exercises and projects created to demonstrate skills in data querying and data analysis.

The examples range from basic SQL queries to more advanced analysis using aggregations, subqueries, and analytical functions.

---

## English

### Objective

Demonstrate knowledge in:

- SELECT statements and filtering  
- Table JOIN operations  
- GROUP BY and aggregations  
- Subqueries  
- Window Functions  
- Data analysis using SQL  

### Repository Structure

**datasets/**  
Datasets used for examples and analysis.

**queries/**  
SQL queries organized by topic.

**projects/**  
Complete data analysis projects using SQL.
