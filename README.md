# sql-data-analysis
# SQL Portfolio

Este repositório contém exercícios e projetos de SQL desenvolvidos para demonstrar habilidades em consulta e análise de dados.

Os exemplos incluem desde consultas básicas até análises mais avançadas utilizando agregações, subconsultas e funções analíticas.

---

## Objetivo

Demonstrar conhecimentos em:

- SELECT e filtros
- JOIN entre tabelas
- GROUP BY e agregações
- Subqueries
- Window Functions
- Análise de dados utilizando SQL

---

## Estrutura do repositório

datasets/  
Bases de dados utilizadas para os exemplos.

queries/  
Consultas SQL organizadas por tema.

projects/  
Projetos completos de análise de dados utilizando SQL.

---

## Tecnologias utilizadas

- MySQL
