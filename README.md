# sql-data-analysis
# SQL Portfolio

##ENGLISH

This repository contains SQL exercises and projects developed to demonstrate skills in data querying and analysis.

The examples range from basic SQL queries to more advanced analyses using aggregations, subqueries, and analytical functions.

---

## Objective

Demonstrate knowledge in:

* SELECT statements and filtering
* Table JOIN operations
* GROUP BY and aggregations
* Subqueries
* Window Functions
* Data analysis using SQL

---

## Repository Structure

**datasets/**
Datasets used for examples and analysis.

**queries/**
SQL queries organized by topic.

**projects/**
Complete data analysis projects using SQL.
